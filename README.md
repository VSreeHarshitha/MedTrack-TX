# MedTrack-TX: Skin Lesion Segmentation

## Overview

MedTrack-TX is a deep learning-based medical image segmentation system designed to identify skin lesion regions from dermoscopic images using the ISIC 2018 dataset.

## Features

- Skin lesion segmentation
- U-Net architecture
- PyTorch implementation
- GPU accelerated training
- Medical image preprocessing

## Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- U-Net
- Google Colab

## Dataset

ISIC 2018 Skin Lesion Segmentation Dataset

## Workflow

1. Dataset Loading
2. Image Preprocessing
3. U-Net Training
4. Loss Computation
5. Model Optimization
6. Segmentation Prediction
